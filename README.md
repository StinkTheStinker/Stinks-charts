# QnA:

"How do I download this?", click on the green button that says 'Code', and click 'Download ZIP'. Should download.

"Can I use your charts?", no.

"Can I use your scripts?", just don't claim it as yours, and also credit me.

"What is stink engine?", modified version of JS Engine made by Jordan Santiago. Engine is very good for modding, even though theres no custom subStates/States you can do like Psych Engine. Also, JS Engine is a fork from Psych Engine, thats why it barely has the latest features of Psych Engine.

"How can I get rid of things that can cause seizures/epilepsy?", you can't sadly, I don't know how to make that one option from the graphics menu affect lua coding. Sorry for the people that can get Epilepsy.

"Did you code every single chart that you made in this?", yes, but no at the same time, im still learning lua coding, Jordan Santiago helps me with lua coding a bit.

"Why is Stink Engine (kinda) the same as JS Engine?", one, I can't do haxe coding that well (that will take a while), two, same answer as I said in the 4th question.

"Are you going to credit the creators?", of course I would, some credits are in scripts, a couple in the github credits.
